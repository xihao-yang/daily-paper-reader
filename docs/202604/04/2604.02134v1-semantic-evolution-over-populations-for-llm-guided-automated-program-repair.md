## 摘要
大型语言模型（LLMs）近年来在自动程序修复（APR）领域展现出强大的潜力，尤其是在通过迭代优化生成和改进候选补丁方面。然而，现有最先进的基于LLM的迭代优化APR方法仍无法充分解决多项挑战，包括：保持修复假设的有益多样性、识别语义相关的修复族、组合互补的部分修复、利用结构化的失败信息，以及摆脱结构上存在缺陷的搜索区域。本文提出了一种面向APR迭代优化的人群语义进化框架——EvolRepair，将基于LLM的APR过程形式化为语义进化算法。EvolRepair重新设计了经典遗传算法在APR任务中的搜索范式，用由LLM和结构化执行反馈支持的语义感知组件替代了传统的语法级操作符。候选修复方案被组织为行为一致的群组，使算法能够保持多样性、对修复族进行推理，并通过跨群体重组互补的修复洞见来合成更强的候选方案。通过利用结构化的失败模式引导搜索方向，EvolRepair既能持续优化有潜力的修复策略，也能在必要时转向替代抽象。实验结果表明，EvolRepair在修复效果上显著优于现有的基于LLM的APR方法。

## Abstract
Large language models (LLMs) have recently shown strong potential for automated program repair (APR), particularly through iterative refinement that generates and improves candidate patches. However, state-of-the-art iterative refinement LLM-based APR approaches cannot fully address challenges, including maintaining useful diversity among repair hypotheses, identifying semantically related repair families, composing complementary partial fixes, exploiting structured failure information, and escaping structurally flawed search regions. In this paper, we propose a Population-Based Semantic Evolution framework for APR iterative refinement, called EvolRepair, that formulates LLM-based APR as a semantic evolutionary algorithm. EvolRepair reformulates the search paradigm of classic genetic algorithm for APR, but replaces its syntax-based operators with semantics-aware components powered by LLMs and structured execution feedback. Candidate repairs are organized into behaviorally coherent groups, enabling the algorithm to preserve diversity, reason over repair families, and synthesize stronger candidates by recombining complementary repair insights across the population. By leveraging structured failure patterns to guide search direction, EvolRepair can both refine promising repair strategies and shift toward alternative abstractions when necessary. Our experiments show that EvolRepair substantially improves repair effectiveness over existing LLM-based APR approaches.

- **现有挑战**：  
  尽管迭代式LLM修复（如ChatRepair、REx）已显著提升修复效果，但仍存在关键问题：
  1. 缺乏 **多样性保持机制**，导致修复过程容易陷入单一假设。

  4. 对测试反馈的利用仅限于“通过率”指标，**未能结构化利用失败信息**。
  5. **难以跳出局部最优** 的修复方向，导致算法陷入错误抽象。

- **研究动机**：  
  作者提出将LLM驱动的APR提升为**群体级的语义进化过程**，借鉴遗传算法的思想，但用语义驱动的操作符取代传统的语法操作符，从而构建更具探索性与泛化性的修复流程。

## 二、方法论：EvolRepair 框架

### 1. 核心思想
- 将LLM引导的APR形式化为**语义进化算法（Semantic Evolutionary Algorithm）**；
- 在每一代中维护一个候选补丁“群体”，通过语义反馈进行选择、交叉与变异；
- 通过**行为聚类（behavioral grouping）**维护多样性；
- 通过**语义变异（semantic mutation）**与**群体级重组（population-level recombination）**实现互补修复集成。

### 2. 主要算法流程  
算法 1（伪代码摘要）：
1. **初始化**：  
   利用LLM生成一组初始修复候选（初始种群）。  
2. **评估与分组**：  
   按测试通过情况构建“行为签名”（即每个补丁通过的测试子集），并基于Jaccard相似度聚类为若干行为组。
3. **交叉（Recombination）**：  
   - 对每个候选组及跨组样本池调用LLM生成新补丁；  
   - LLM的prompt明确要求整合各候选中“正确部分”生成更优方案。
4. **变异（Mutation）**：  
   - 根据失败测试的反馈，用结构化的错误报告指导LLM进一步修改当前候选。
5. **选择（Selection）**：  
   - 移除重复候选，并按通过率与代码长度排序筛选出下一代。  
6. **终止条件**：  
   若某候选通过全部测试，即认为修复完成。

### 3. 数学建模概述
- **目标函数**：最大化修复通过率  
  $$\max_{P' \in Y} F(P') = \frac{1}{M}\sum_{j=1}^{M} I[Eval(P',t_j) = 1]$$  
- **行为签名（Behavioral Signature）**：  
  $$\psi(P') = \{ j | Eval(P',t_j) = 1 \}$$  
  表示候选修复通过的测试集合，用于语义聚类与相似度计算。

- **群体更新机制**：  
  $$P^{(g+1)} = TopN(U^{(g)})$$  
  其中 \(U^{(g)}\) 为当前种群的合并与重采样集合。

## 五、实验结果与主要发现

1. **整体性能（RQ1）**  
   - EvolRepair 在所有模型上性能最优。  
   - 以 DeepSeek V3.1 为例：  
     - Pass@1 = 96.63%，Pass@3 = 98.18%；  
     - 相比 REx 提升约 +2%，相比 ChatRepair 提升 +3%。

2. **行为覆盖（RQ2）**  
   - EvolRepair 的平均覆盖率 (TCC) 优于其他方法，且“覆盖差距Δ”最小；
   - 说明其能更好地整合不同候选的修复行为，避免局部最优。

3. **语义交叉有效性（RQ3）**  
   - “组合率”指标显示：约 20–60% 的交叉补丁能整合多父代的独立部分修复；
   - 证明群体级重组确实能合成更强的新补丁。

4. **消融实验（RQ4）**  
   - 去除语义变异或交叉会显著性能下降（Pass@1降至88.11%或94.50%）；
   - 语义聚类优于基于适应度或随机分组。

5. **超参数灵敏度（RQ5）**  
   - 参数调整（群体规模、变异次数）对性能影响不大（变化<1%），表明方法稳定。
