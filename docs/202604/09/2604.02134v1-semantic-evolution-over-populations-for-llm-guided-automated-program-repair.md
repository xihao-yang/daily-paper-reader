## 摘要
大型语言模型（LLMs）近年来在自动程序修复（APR）领域展现出强大潜力，特别是通过迭代优化产生并改进候选补丁。然而，最先进的基于 LLM 的迭代优化 APR 方法仍难以充分应对以下挑战：保持修复假设之间的有用多样性、识别语义相关的修复族、组合互补的部分修复、利用结构化失败信息以及逃离结构缺陷的搜索区域。本文提出了一种用于 APR 迭代优化的基于种群的语义进化框架，称为 EvolRepair，将基于 LLM 的 APR 表述为一种语义进化算法。EvolRepair 重新构建了经典遗传算法在 APR 场景中的搜索范式，但用由 LLM 和结构化执行反馈驱动的语义感知组件替代了其基于语法的算子。候选修复被组织为行为一致的分组，从而使算法能够保持多样性、对修复族进行推理，并通过在种群中重组互补的修复见解来综合出更强的候选方案。通过利用结构化的失败模式引导搜索方向，EvolRepair 既能细化有前景的修复策略，也能在必要时转向替代抽象。实验结果表明，EvolRepair 在修复有效性方面显著优于现有的基于 LLM 的 APR 方法。

## Abstract
Large language models (LLMs) have recently shown strong potential for automated program repair (APR), particularly through iterative refinement that generates and improves candidate patches. However, state-of-the-art iterative refinement LLM-based APR approaches cannot fully address challenges, including maintaining useful diversity among repair hypotheses, identifying semantically related repair families, composing complementary partial fixes, exploiting structured failure information, and escaping structurally flawed search regions. In this paper, we propose a Population-Based Semantic Evolution framework for APR iterative refinement, called EvolRepair, that formulates LLM-based APR as a semantic evolutionary algorithm. EvolRepair reformulates the search paradigm of classic genetic algorithm for APR, but replaces its syntax-based operators with semantics-aware components powered by LLMs and structured execution feedback. Candidate repairs are organized into behaviorally coherent groups, enabling the algorithm to preserve diversity, reason over repair families, and synthesize stronger candidates by recombining complementary repair insights across the population. By leveraging structured failure patterns to guide search direction, EvolRepair can both refine promising repair strategies and shift toward alternative abstractions when necessary. Our experiments show that EvolRepair substantially improves repair effectiveness over existing LLM-based APR approaches.

## 一、核心问题与研究背景

- **研究主题**：本文研究如何利用大型语言模型（LLMs）提升自动程序修复（Automated Program Repair, APR）的效果。  
- **背景与挑战**：
  - 传统 APR 方法（如 GenProg、SemFix、Nopol）主要基于语法编辑或符号约束求解，容易出现过拟合、搜索空间受限等问题；
  - 深度学习及 LLM 的兴起使得代码修复可视作“有条件的代码生成”任务，具备更强的修复语义能力；
  - 然而，当前主流 LLM 迭代修复模型（如 REx、ChatRepair）仍存在五大问题：
    1. 难以保持候选修复的多样性；
    2. 无法识别语义相关的修复族；
    3. 缺乏跨候选组合互补部分修复的能力；
    4. 仅基于测试通过率的反馈，无法利用结构化失败信号；
    5. 难以跳出语义上错误的局部搜索区域。
- **研究目标**：提出一种新的语义进化式的群体优化框架，使 LLM 驱动的修复过程能整合多样候选、进行语义层组合并自适应调整搜索方向。

## 二、方法论与算法流程

### 1. 核心思想
- **EvolRepair 框架**：将 LLM 引导的 APR 过程表述为一个“语义进化算法（Semantic Evolutionary Algorithm）”。
- 与传统遗传算法相比：
  - 以 **语义为中心** 而非语法编辑；
  - 使用 LLM 作为“语义变异算子（semantic mutation operator）”；
  - 引入“行为分组（behavioral grouping）”与“群体水平重组（population-level recombination）”机制。

### 2. 关键技术组件
- **群体维护**：保留多个候选修复 \( P^{(g)} = \{P_1^{(g)}, ..., P_N^{(g)}\} \)。
- **行为签名**：定义每个修复的行为 \(\psi(P') = \{j | Eval(P', t_j) = 1\}\)，表示该修复通过的测试索引集。
- **适应度函数**：
  - 采用测试通过率：  
    \( F(P') = \frac{1}{M} \sum_{j=1}^{M} I[Eval(P', t_j) = 1] \)。
- **行为分组与聚类**：
  - 按通过测试的集合进行 Jaccard 相似度聚类；
  - 群体被划分为若干“语义相似修复族”用于后续重组。
- **交叉重组（Recombination）**：
  - LLM 接收多个候选（同组或跨组）代码及其测试表现，综合其正确逻辑生成新的候选修复；
  - 属于“群体级别的语义交叉”，而非传统的两父母式语法交叉。
- **语义变异（Semantic Mutation）**：
  - 选取表现较好的候选，根据失败测试反馈信息（输入、期望输出、错误信息）进行有针对性的改写。
- **演化流程（Algorithm 1）**：
  - 初始化候选群体；
  - 评估并分组；
  - 进行跨组采样；
  - 执行群体重组与个体语义变异；
  - 选择 Top-N 候选形成下一代；
  - 若发现完全正确修复则提前终止。
